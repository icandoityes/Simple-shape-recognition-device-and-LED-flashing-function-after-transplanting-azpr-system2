1、项目概要：
（1）、简单的图形识别器组成以SEA-S7为开发板卡，以摄像头模组为识别装置，最后通过显示器来进行识别后的视频传输。实现的功能有视频传输、简单图像如三角形物体、正方形物体、圆形物体的形状识别。
（2）、AZPR片上系统移植和简单的流水灯闪烁功能的显示。作品组成为SEA-S7开发板一块。实现功能：具有azpr片上系统所定义的时钟、LED、管脚拓展等功能，展示功能为LED灯闪烁功能。系统实现过程为azpr片上系统的移植（这里引用课程中赵老师的讲课内容），移植后的程序修改，在vivado上进行程序的综合，最后下载到板卡上进行功能的验证。
